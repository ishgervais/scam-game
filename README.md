# Command Line Interface Game


### How to play these games

1. Clone this repository

```bash
git clone https://github.com/ishgervais/scam-game.git
cd scam-game
```

2. Install requirements

```bash
pip install -r requirements.txt
```

3. Start Game

```bash
python3 playgame.py
```
